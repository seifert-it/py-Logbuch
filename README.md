# Python Logbuch (CLI)

Ein kleines, bewusst überschaubares **Python-Übungsprojekt** in Form eines
kommandozeilenbasierten Logbuchs.

Das Projekt dient dem Lernen und Anwenden von:
- Dateiverarbeitung (CSV)
- Parsing von Textdaten
- Datum- und Zeitverarbeitung mit `datetime`
- Kontrollstrukturen, Funktionen und sauberer Programmstruktur
- einfacher Suche und Statistik in der Konsole

---

## Features

 Einträge hinzufügen (Datum/Uhrzeit automatisch)
 Alle Einträge anzeigen
 Suche nach:
  - Kategorie
  - Text
  - Kategorie + Text
  - Datum
  - Zeitraum von–bis
Statistik:
  - Einträge pro Kategorie
  - Einträge pro Monat
  - ASCII-Mini-Charts in der Konsole

---

## Voraussetzungen

- Python **≥ 3.10**
- Keine externen Abhängigkeiten (nur Standardbibliothek)
